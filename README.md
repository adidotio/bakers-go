# BakersGo

BakersGo is a modern, complex React application created to serve as a comprehensive testing ground for the **MultiLipi Dashboard's** translation capabilities.

## Features

- **Massive Mock Data:** Over 20 diverse bakery products, each with long descriptions, ingredients, allergens, and nutritional info.
- **Complex UI Components:** Including product filtering, multi-step custom cake order forms, interactive FAQs, and blog articles.
- **Modern Styling:** Built with Tailwind CSS v4 featuring a warm, premium bakery aesthetic, complete with hover animations and complex grid layouts.
- **Deep Routing:** Multiple distinct pages that require routing logic and separate component translations.

## Running the Project

```bash
cd bakers-go
npm install
npm run dev
```

Visit `http://localhost:5173` to explore the dummy website.
